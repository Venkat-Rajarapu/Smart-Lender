## Project Structure

This repository follows the standard AI/ML/Gen-AI project lifecycle, organized into 8 phases:

1. **Brainstorming & Ideation** — Problem identification, idea exploration, and initial concept notes.
2. **Requirement Analysis** — Functional and non-functional requirements, technical feasibility.
3. **Project Design Phase** — Entity Relationship Diagram, data flow design, and application architecture.
4. **Project Planning Phase** — Project workflow, milestones, and task breakdown.
5. **Project Development Phase** — Data collection, EDA, preprocessing, model building, and application building.
6. **Project Testing** — Testing the application, models, and integration.
7. **Project Documentation** — Final project report and documentation.
8. **Project Demonstration** — Demo video / final walkthrough of the working application.
